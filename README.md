# debian-home-server

## Setting up Debian

Start SSH Server by running commands in `pre.sh`

Copy `git.sh` to Debian

Run `sudo ./git.sh <git-username> <git-email>`

Navigate to `git/debian-home-server/setup-scripts`

Run `sudo install.sh`

Run `sudo auto-start.sh`
